# jmbtechnical.com

Business website for JMB Technical - Industrial Automation & Controls Engineering.

## Stack

- [Astro](https://astro.build) - Static site generator
- [Tailwind CSS](https://tailwindcss.com) - Utility-first CSS
- [Cloudflare Pages](https://pages.cloudflare.com) - Hosting & CDN

## Pages

- **Home** - Hero, services overview, featured projects, industries served
- **About** - Background, career timeline, technology stack
- **Services** - PLC programming, SCADA/HMI, robotics, custom software
- **Projects** - Portfolio with GitHub links
- **Blog** - Markdown-based articles
- **Contact** - Contact form via FormSubmit

## Development

```sh
npm install
npm run dev        # localhost:4321
npm run build      # production build to ./dist/
```

## Deployment

Automatically deployed to Cloudflare Pages on push to `main`.
